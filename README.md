# coffee_sales_dashboard

## Objectives:
1) Monitor overall business performance using KPIs.
2) Analyze sales trends over time.
3) Compare actual sales with target sales.
4) Identify top-performing products.
5) Evaluate profit across different markets.
6) Provide interactive filtering for business analysis.

## Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Total Margin
- Target Sales
- Target Profit

## Tools Used: 
1) Microsoft Power BI Desktop

## Key Insights:
1) Sales performance is tracked across multiple years to identify growth trends.
2) Product Line analysis highlights the contribution of Beans and Leaves to overall sales.
3) Profit comparison across markets helps identify high-performing regions.
4) Treemap visualization quickly identifies the best-selling products.
5) Actual vs Target Sales comparison highlights areas exceeding or missing business targets.

##  Dataset
The dashboard is built using a Coffee Chain sales dataset taken from Kaggle
